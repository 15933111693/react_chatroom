# 如何启动它?

首先进行安装依赖和开发依赖,之后开启服务

```sh
$ cd react_chatroom
$ npm i
$ npm start
```

如果您用的是yarn

```sh
$ cd react_chatroom
$ yarn
$ yarn start
```
